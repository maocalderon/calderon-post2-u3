## Descripción del Proyecto
Este proyecto consiste en el diseño e implementación de un dashboard web responsivo. Se utiliza **CSS Grid** para la arquitectura general de la página (layout principal) y **Flexbox** para la alineación de componentes internos y elementos de navegación. El desarrollo se realizó sin el uso de frameworks externos para demostrar el dominio de los modelos de layout modernos de CSS3.

## Características Técnicas
- **Layout:** Implementación de `grid-template-areas` para definir sidebar, topbar y main.
- **Responsividad:** Uso de `repeat(auto-fill, minmax())` para las tarjetas de estadísticas y Media Queries para adaptar el contenido a dispositivos móviles (768px).
- **Componentes:** Sidebar con Flexbox en columna, Topbar con distribución `space-between` y tablas de datos con badges personalizados.
- **Sin Frameworks:** Estilos 100% personalizados siguiendo buenas prácticas de CSS3.
